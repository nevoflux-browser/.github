# NevoFlux Browser

**Don't Just Browse. Command.**

NevoFlux is an AI-native browser with intelligence built into the engine — not bolted on as an extension. It gives your AI companion identity, memory, and progressive autonomy: from chat, to browser control, to full system automation.

### Highlights

- 🧠 **AI with Identity** — A companion that evolves its personality, remembers across sessions, and adapts to you
- 🎛️ **Progressive Autonomy** — Three modes: Chat → Browser Control → Agent (full system automation)
- 🔌 **16+ LLM Providers** — Claude, GPT, Gemini, DeepSeek, Qwen, local Ollama, and more
- 🖥️ **70+ Automation APIs** — Browser control, screen capture, mouse & keyboard across web and OS
- 🧩 **Canvas Micro Apps** — Generate dynamic apps with React, Vue, or Svelte on the fly
- 🔗 **OpenClaw Integration** — 50+ service connectors for email, calendar, tasks, smart home
- 🔒 **Privacy-First** — Open source, local-first, encrypted memory
- 🦊 **Built on Firefox/Zen** — MPL-2.0 licensed, runs on Windows, macOS & Linux
